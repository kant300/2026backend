버튼은 이벤트처리하기 위해서 사용
    html  : input 태그에 type을 button
    html5 : button 태그를 지원

    버튼을 이용해서 서버에 값을 전달할 수 있다.
    버튼을 이용해서 javascript로 처리해서 서버에 값을 전달할 수 있다.

    버튼의 용도
    type = "button" : 일반버튼
    type = "reset" : 입력영역에 있는 내용을 모두 초기화
    type = "submit" : 입력영역에 있는 모든 내용을 서버에 전달

    button이 없으면 클라이언트(브라우저,사용자)와 서버(스프링부트,운영)간에 정보전달이 불가능
