문자열와 메소드

모든언어에서 문자열 데이터형이 존재하지 않는다.
프로그램으로 문자열을 처리하도록 만들어서 라이브러리 제공한다.
문자열처리를 위한 라이브러리가 String클래스

1. 문자열과 함께 사용하는 주요 메소드
 length() : 문자열의 길이   String a="abc"; ==>3
 isEmpty() : 빈문자열인지 판별 String a=""; ===>true, String a= " "; ===>false
 isBlank() : 빈문자열인지 판별 String a=""; ===>true, String a= " "; ===>true
 equals() : 저장된 문자열과 같은지 비교 a.equals("abc") a에 내용이 abc와 같으면 true
       String a ="Abc" a.equals("Abc") ->true, a.equals("abc) ->false  
       대소문자구분해서 비교
 equalsIgnoreCase() : 저장된 문자열과 같은지 비교
       String a ="Abc" a.equalsIgnoreCase("Abc")->true, a.equalsIgnoreCase("abc")->true
       대소문자구분없이 비교

==javascript에 많이 사용==
 charAt(위치) : 지정한 위치에 문자를 읽어온다.(문자열을 배열처럼)
       String a="ABC";  a.charAt(0) ==> A a.charAt(1) ==> B
 indexOf("문자") : 해당하는 문자의 위치값(앞->뒤로 검색), 첫번째 검색시 종료
       String a="AbcAbc";  a.indexOf("A") =>0
 lastIndexOf("문자") : 해당하는 문자의 위치값(뒤->앞 검색), 첫번째 검색시 종료
   String a="AbcAbc";  a.lastIndexOf("A") =>3
 contains("문자열") : 해당문자열이 존재하는지 판별
   String a="abctestlove" a.contains("love") =>true a.contains("es")=>true
   ==>검색기능(주어진 검색어가 존재하는 데이터만 조회)
 startWith/endWith :문자열에 앞이나 뒤에 해당문자열이 포함되어 있는지 판별
   String a="abctestlove" a.startWith("ab") =>ture, a.endWith("ab") => false
 subString(시작, 끝) : 문자열에 지정된 위치에서 지정된 위치까지 문자열을 추출
   String a="abctestlove" a.subString(1,4) => "bct"
 replace("찾을문자열", "바꿀문자열") : 지정문자열을 받아 새로운 문자열로 치환
   String a="abctestlove" a.replace("ab","yz") => yzctestlove
