연산자
계산을 담당한다.

1.연산자 종류
산술연산자 : 숫자연산
관계연산자 : 두값의 관계비교연산
논리연산자 : 관계연산자들의 관계
비트연산자 : 2진수의 값을 연산(화면이동, 화면지우기)

2. 산술연산자(숫자를 연산하는게 목적)
+ :덧셈
- :뺄셈
* :곱셈
/ :나눗셈(몫)
% :나눗셈(나머지)
**(^) :제곱

증감연산자
++ : 변수에 값을 1증가
-- : 변수에 값을 1감소

대입연산자
+= : 변수에 값을 더해서 변수에 다시 저장
-= : 변수에 값을 빼서 변수에 다시 저장
*= : 변수에 값을 곱해서 변수에 다시 저장
/= : 변수에 값을 나누어서 변수에 다시 저장
%=
**=

3. 관계연산자(두개의 값의 관계를 비교해서 true(맞으면), false(틀리면)결과 받는다)
관계연산자   의미          설명
>           크다          앞에 값이 뒤에 값보다 크다.(30>20) 
<           작다          앞에 값이 뒤에 값보다 작다.(20<30)
>=          크거나같다     앞에 값이 뒤에 값보다 크거나 같다.(30>=20, 20>=20)  
<=          작거나같다     앞에 값이 뒤에 값보다 작거나 같다.(20<=30, 30<=30)  
==          같다          앞에 값이 뒤에 값과 같다.(30==30) javascript에서는 ===
!=          같지않다       앞에 값이 뒤에 값과 같지 않다.(20!=30) 

4. 논리연산자(두개이상의 조건(관계연산자)을 비교해서 true(맞으면), false(틀리면)결과 받는다)
             논리곱(AND) 논리합(OR)  부정(NOT)
A       B       A&&B    A||B       !A
true    true    true    true       false    &&는 모두조건이 true이면 결과는 true
true    false   false   true       false    ||는 하나의 조건이라도 true이면 결과는 true 
false   true    false   true       true      !는 조건의 결과를 반대로 변경
false   false   false   false      true
                       1(true) &&(*) 0(false) =  
(20>10) && (30>10) :  true && true => true
(20>10) && (10>30) :  true && false => false
(10>20) && (30>10) :  false && true => false
(10>20) && (10>30) :  false && false => false
                        1(true) ||(+) 0(false) = 
(20>10) || (30>10) :  true || true => true  1+1=1
(20>10) || (10>30) :  true || false => true 1+0=1
(10>20) || (30>10) :  false || true => true 0+1=1 
(10>20) || (10>30) :  false || false => false 0+0=0

!(20>10) : !true=>false
!(10>20) : !false=>true

5. 우선순위(연산을 할때 계산하는 순서)
1) (), []
2) +,-(부호), ++, --, !
3) *,/,%
4) +,-
5) ==, !=
6) &&
7) ||
8) +=,-=,*=,/=,%=
