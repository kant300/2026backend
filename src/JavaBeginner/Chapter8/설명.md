배열(Sprintboot보다는 javascript에 많이 사용)
하나의 변수명에 여러값을 저장하는 방법
선언시 : int [전체수] ---저장할 전체 갯수
사용시 : 변수[위치] ---위치(첨자)


0부터 시작(visual basic은 1부터 시작)

선언할 때 [] 개수에 따라서 다차원배열
1차원배열 int [열]
2차원배열 int [행][열]
3차원배열 int [면][행][열]

1. 선언방법(전체사용할 크기를 지정)
   데이터형에 선언 : int [4] a;
   변수에 선언 : int a[4];
   생성자를 이용한 선언 : int [] a = new int[4];

 Springboot에서 배열대신 맵, 리스트를 주로 사용 - 1차원 주로 사용
 javascript, react에서는 배열사용(맵, 리스트 기능이 없다.) - 1차원 주로 사용
 python에서는 리스트, 튜플, 딕셔너리를 주로 사용 - 2차원 주로 사용 - 빅데이터

2. 초기값 설정방법
   1차원  int [열] a = {값, 값, 값};
   2차원  int [행][열] a = {{열값,열값,열값}, 
                           {열값,열값,열값}, 
                           {열값,열값,열값}}
   3차원 int [면][행][열] a= {
                             {{열값,열값,열값},
                              {열값,열값,열값},
                              {열값,열값,열값}}, 
                             {{열값,열값,열값},
                              {열값,열값,열값},
                              {열값,열값,열값}},
                             {{열값,열값,열값},
                              {열값,열값,열값},
                              {열값,열값,열값}}}
