조건문
1. 일반조건문(맞는 경우에만 처리)
 if(조건) {
   조건을 맞을 때 처리
 }

2. 분기조건문(2가지 조건에 따른 처리)
 if(조건) {
    조건을 맞을 때 처리
 } else {
    조건이 틀릴 때 처리
 }

3. 중첩조건문(AND연산과 유사) - 밖과 안의 조건 내용이 다른 경우
 if(조건) {  성별
   if(조건) { 나이
   } else {
   }
 } else {
   if(조건) {
   } else {
   }
 }

4. 다중조건문(동일한 내용으로 여러 조건을 처리)
 if(조건) {
    맞을때 처리
 } else if(조건) {
    맞을때 처리
 } else {
    모든 조건이 틀리면
 }

5. switch문(값이 같은 경우를 비교)
  switch(변수) {
    case 비교값:
        break; //처리 종료
      :
    default:
      만족하는 조건이 없을 때
  }
