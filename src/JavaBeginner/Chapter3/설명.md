변수와 데이터 형식

1. 변수, 상수, 리터럴, 예약어
변수 : 반드시 사용, 값을 메모리에 저장해서 필요할 때마다 불러서 사용, 변수하는 수(저장공간)
상수 : 특정한 값을 단어화시켜서 사용, 변하지 않는 값을 지정해서 사용
리터럴 : 변수에 저장할 수 있는 모든 종류의 값, (문자, 문자열, 숫자, 논리값)
예약어 : Java에서 이미 사용하는 단어(사용자가 임의적으로 사용이 불가능)

2. 변수의 규칙
영문자로 구성, $, _ 문자로 인식, 띄어쓰기 불가능, 예약어 사용불가능
=>일반적으로는 값의 성격에 맞게 이름을 지정합니다.(국어점수 50점==> kor=50)

3. 변수의 종류(변수를 사용할 수 있는 범위)
전역변수(멤버변수) : class내에 선언된 변수
지역변수 : 메소드나 { }안에서 선언된 변수..메소드나 {}를 벗어나면 소멸

4. 데이터형식(변수에 저장할 수 있는 값의 형태)-데이터형, 데이터타입
자바프로그램 개발시
short, int, long : 정수형 숫자(소숫점이 없는 숫자)(1, 100, 30)
float, double : 실수형 숫자(소숫점이 있는 숫자)(12.23)
char : 한 문자('a') ''안에 문자를 입력
String : 문자열("abc") ""안에 문자를 입력
boolean : 논리형(참, 거짓) - Ture(ture, TRUE), False(false, FALSE)-언어별로 표현식이 다름

스프링부트 개발시(래퍼클래스 데이터형)
Short, Integer, Long : 정수형 숫자
Float, Double : 실수형 숫자
Character : 한문자
String : 문자열
Boolean : 논리형
LocalDate : 날짜형
LocalDateTie : 날짜/시간형

5. 변수선언방법
데이터형 변수명; -선언만    int sum;
데이터형 변수명 = 값; -초기값 설정   int sum=5;

데이터형 변수명; -초기값 설정  int sum;
변수명 = 값;                sum = 0; 변수에 값만 저장할 때(초기값)

6. 변수값을 저장 및 확인하는 방법
저장 : Scanner를 입력해서 키보드로 입력
확인 : System.out.println(), System.out.print(), System.out.printf()로 모니터에 출력


진법변환
10진수->2진수,8진수,16진수 변환

24(10) - (2), (8), (16) : 원하는 진수로 나눈 나머지값을 역으로 읽어서 처리

     24                 24             24
   2|12 ---0 나머지값  8| 3 ---0      16| 1---8
   2| 6 ---0          8| 0 ---3      16| 0---1
   2| 3 ---0           30(8)           18(16)
   2| 1 ---1
   2| 0 ---1
   11000(2)

(2), (8), (16) -  (10) : 각 자리의 진수의 제곱의 합으로 구한다.
  2^4 2^3 2^2  2^1  2^0

-----------------------

   8^1   8^0                16^1   16^0
   8      1                  16      1
