반복문

일정한 범위를 지정한 횟수만큼 반복적으로 처리
조건을 만족하는 동안에 반복을 수행

1. 기본 for문(초기값, 조건, 증감 3개로 구성이 되는데 상황에 맞게 적용)
   for(초기값; 조건; 증감) { 

   }
   
    초기값;
   for(;조건;증감) {

   }
   
   for(초기값;조건;) {
      증감;
   }

   for(초기값;;증감){
       조건;
   }

   for(;;) {
       무한반복
   }

2. 중첩 for문(2개이상의 for문을 사용하는 경우)
   for(초기값; 조건; 증감) {
       for(초기값; 조건; 증감) {
     
       }
   }

   중첩 for문은 밖에 for문과 안쪽에 for문의 반복수를 정확히 계산
   for(int i=0; i<3; i++) {
      for(int j=0; j<3; j++) {
      
      }
    }
             i의 반복              j의 반복
i초기값 0     0          j초기값 0  0, 1, 2
             1          j초기값 0  0, 1, 2
             2          j초기값 0  0, 1, 2
      지정한 반복수만큼    매번 초기화후 지정한 반복수만큼
             3           i*j= 9
