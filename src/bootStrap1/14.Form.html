서버와 클라이언트간에 데이터를 전달하는 방식
    Controller 방식 : get방식(조회,삭제), post방식(삽입,수정) - 작업완료후 페이지로 이동
    Rest 방식 : get방식(조회), post방식(삽입), put방식(수정), delete방식(삭제) - 작업완료후 메시지

    form : 입력작업의 범위를 지정, form안에 있는 내용들이 서버에 전달이 됩니다.
      method : 서버에 전달하는 방식(post방식, get방식)
      action : 서버에 요청할 주소(전달할 주소)
    .form-label : form 디자인 레이블
    .form-control : form에 존재하는 input 태그의 디자인
    .form-check, form-check-input : 체크상자 디자인
    .for="id명" : label(제목)-input(입력폼) label에 해당하는 input태그의 id명
    name : 변수명, value : 초기값, placeholder : 안내문구
