버튼 : 브라우저에 작업한 내용 및 작업을 서버에 전달할 때
    .btn : 버튼의 모양
    .btn-색상 : 버튼의 색상을 적용(배경)
    .btn-outline-색상 : 버튼의 테두리 색상을 적용
    .btn-sm, btn-md, btn-lg : 버튼의 크기지정
    .btn-block : 브라우저 크기만하게(부모태그에 d-grid로 선언)

    href : 클릭시 이동할 주소, #은 업음

    .btn-group : div에서 선언하면 div안에 버튼들을 그룹화시켜서 출력(메뉴)
    .btn-group-vertical : 세로메뉴(서브메뉴)
