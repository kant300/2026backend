태그명 -> tab을 클릭하면 자동 태그생성

    색상은 대상-색상명-명도(투명도) , 대상-bg-색상명-명도(투명도)
    .text(button)-primary : 동작용(긍정적인 응답) -"저장"
    .text(button)-secondary : 보조용 - "취소"
    .text(button)-info : 사용자에게 안내문구를 제공할 때 - "도움말"
    .text(button)-warning : 사용자에게 경고문구를 제공할때
    .text(button)-danger : 사용자에게 위험문구를 제공할때
    .text(button)-white : 흰색
    .text(button)-dark : 검정색
    .text(button)-muted : 비활성화

    색상명은 공통
