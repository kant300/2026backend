카드 : 둥근테두리로 영역을 구현한 태그
                header : 카드의 제목, content : 카드의 본문내용, footer : 동작버튼
        .card : 카드형태, 부모태그(div)
            .card-header :
            .card-content :
            .card-footer :
        .bg-색상 :
        .card-img-top : img태그에서 카드에 이지를 배치(위쪽)
        .card-img-bottom : 아래쪽
        .card-img-overlay : 이미지와 본문을 겹치기

        m-값(1~5) : 바깥쪽 여백, 현재 태그에 위,아래, 좌,우 태그와의 거리, 상하좌우전체 적용
            mt-값 : 위, ms-값 : 왼쪽, me-값 : 오른쪽, mb-값 : 아래쪽
        p-값(1~5) : 안쪽 여백 , 현재 태그와 안에 여백, 상하좌우전체 적용
            pt-값 : 위, ps-값 : 왼쪽, pe-값 : 오른쪽, pb-값 : 아래쪽
